#Naming a variable:
Rules for naming python variables:
--> A variable name  must start with  letter or the underscore character
--> A variable name cannot strat with number 
--> A variable name can only contain alpha-numeric characters and undersores  (A-z, 0-9, and _)
--> variable names are case-sensitive(age, Age and AGE are three different variables)
--> Inbuilt keywords shoult not be used as identifiers
